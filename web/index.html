BridgeClaw Calendar App

Telegram-first calendar operations with a Convex backend, explicit confirmation, and daily 7-day digests.

System

Status
Checking
Backend
Convex health route
UI
Static Render shell

Capabilities

  • Telegram webhook boundary
  • Google OAuth boundary
  • Daily digest formatter
  • Pending mutation gate

Deployment map

  1. Create a Convex project and set the environment variables from .env.example.
  2. Deploy /web as a Render Static Site and set APP_BASE_URL to that URL.
  3. Point Telegram webhook traffic to /telegram/webhook/:secret on the Convex deployment.
  4. Run Google OAuth with the Convex callback URL.
